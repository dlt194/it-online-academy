Activity 6.1: Manipulating Arrays

Objective:

Create a mini project to practice JavaScript array methods, focusing on creating, indexing, and using methods like push, pop, shift, unshift, splice, and forEach.

Instructions:
  1. Setup:
    • Create a new folder for your project files.
    • Inside the folder, create index.html, styles.css (optional), and script.js files.
  2. HTML Structure:
    • In index.html, set up the basic structure with a header (<h1>), a container for displaying array elements, and input fields/buttons for adding, removing, and manipulating array elements.
  3. JavaScript Implementation:
    • Setup:
      • Open your script.js file in a text editor to implement JavaScript functions for array manipulation.
    • Initialize the Array:
      • Start by initializing an empty array using let array = [];. This array will store elements that you'll manipulate using different array methods.
    • Implementing Array Methods:
      • Push: Create a function addElement() that retrieves an element from an input field and adds it to the end of the array using array.push(element).
      • Pop: Create a function that removes the last element using array.pop() when a "Remove Last" button is clicked.
      • Shift: Create a function that removes the first element using array.shift() when a "Remove First" button is clicked.
      • Unshift: Create a function that adds an element to the beginning using array.unshift(element) when an "Add First" button is clicked.
      • Splice: Create a function removeElement(index) that removes an element at a specific index using array.splice(index, 1).
      • ForEach: Create a function displayArray() that uses array.forEach() to display each element in the HTML area with ID #arrayElements.
    • Displaying Array State:
      • Use DOM manipulation in displayArray() to update the #arrayElements div with the current state of the array.
      • Format each element like "Element 1: value".
    • Testing and Validation:
      • Test each method to ensure they update the array and UI correctly.
      • Validate user input and handle edge cases like empty arrays or invalid operations gracefully.
  4. Styling (Optional):
    • Use styles.css to enhance visuals and user interaction.
  5. Testing and Validation:
    • Repeat tests for each method after styling to confirm expected behavior.
