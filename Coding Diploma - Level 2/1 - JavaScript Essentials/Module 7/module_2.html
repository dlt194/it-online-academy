Objects, Methods, Properties & Functions

3 of the most common objects in JavaScript are:

  • Window
  • Document
  • Console

Window is the global object in JavaScript, it represents the browser window. The document object represents the HTML document loaded in the browser, and the console object provides access to the browser's debugging console.

To access the window object, you can use the `window` keyword.


window.alert("Hello, World!");

This code will display an alert box with the message "Hello, World!" in the browser.




The window object has various properties. A good example of this would be onload, which is an event handler that triggers when the page is loaded. 


window.onload = function() {
  console.log("Page is loaded!");
};

This code will log "Page is loaded!" to the console when the page finishes loading.

This would be considered an anonymous function. This is because it does not have a defined name, it runs in the background.

Variables & Data Types

Variables are used to store data in JavaScript. They can hold different types of values, such as numbers, strings, booleans, and objects. In JavaScript, you can declare variables using the 'var', 'let', or 'const' keywords.


  • var: Used to declare a variable that can be reassigned later. It has function scope.
  • let: Used to declare a block-scoped variable that can be reassigned. It is preferred over 'var' for block scope.
  • const: Used to declare a block-scoped variable that cannot be reassigned. It is used for constants. 

var myAge;
var myAge = 30;
var x = 10;
var charaterName = "Sponge";
var isRaining = false;

let myAge;
let myAge = 30;
let x = 10;
let charaterName = "Sponge";
let isRaining = false;

const myAge;
const myAge = 30;
const x = 10;
const charaterName = "Sponge";
const isRaining = false;

Conditional Statements

Conditional statements allow you to execute different code based on certain conditions. The most common conditional statements in JavaScript are 'if', 'else if', and 'else'. 


let myName = "Dan";
if (myName === "Dan") {
  // Code to execute if condition is true
  console.log("Hello, Dan!");
} else if (anotherCondition) {
  // Code to execute if anotherCondition is true
  console.log("Hello, someone else!");
} else {
  // Code to execute if none of the conditions are true
  console.log("Hello, stranger!");
}

You can also use 'switch' statements.


switch (myName) {
  case "Dan":
    console.log("Hello, Dan!");
    break;
  case "John":
    console.log("Hello, John!");
    break;
  default:
    console.log("Hello, stranger!");
    break;
}

The 'switch' statement evaluates an expression and executes the corresponding case block based on the value of the expression.

Arrays

Arrays are used to store multiple values in a single variable. They can hold different types of data, including numbers, strings, and objects. In JavaScript, you can create an array using square brackets []. 


let myArray = [1, 2, 3, 4, 5];
let myArray = ["apple", "banana", "orange"];
let myArray = [1, "apple", true, { name: "Dan" }];

You can access elements in an array using their index, which starts at 0. For example, myArray[0] will give you the first element of the array. 


let myArray = [1, 2, 3, 4, 5];
console.log(myArray[0]); // Output: 1
console.log(myArray[1]); // Output: 2
console.log(myArray[2]); // Output: 3

You can also use methods like push() to add elements to the end of the array, pop() to remove the last element, and length to get the number of elements in the array. 


let myArray = [1, 2, 3, 4, 5];
myArray.push(6); // Adds 6 to the end of the array
myArray.pop(); // Removes the last element (5)
console.log(myArray.length); // Output: 5

Arrays are versatile and can be used to store lists, collections, and more complex data structures.

Loops

Loops are used to execute a block of code repeatedly until a specified condition is met. The most common types of loops in JavaScript are 'for', 'while', and 'do...while'.


Below is an example of some code that can easily be simplified with a loop. 


const library = [
  "Lord of the Rings",
  "Harry Potter",
  "The Hobbit",
  "Pride and Prejudice",
  "To Kill a Mockingbird",
  "1984",
]

let text = "";

text = text + library[0] + "<br>";
text = text + library[1] + "<br>";
text = text + library[2] + "<br>";
text = text + library[3] + "<br>";
text = text + library[4] + "<br>";
text = text + library[5] + "<br>";

document.getElementById("demo").innerHTML = text;

This code can be simplified using a loop to iterate through the array and display each book title. 


const library = [
  "Lord of the Rings",
  "Harry Potter",
  "The Hobbit",
  "Pride and Prejudice",
  "To Kill a Mockingbird",
  "1984",
];
let text = "";
for (let i = 0; i < library.length; i++) {
  text += library[i] + "<br>";
}
document.getElementById("demo").innerHTML = text;

In this example, the 'for' loop iterates through each element in the 'library' array and appends it to the 'text' variable, which is then displayed in the HTML element with the ID "demo".
