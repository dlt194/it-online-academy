A Deeper Look at Arrays

Arrays are a special type of object in JavaScript that allow you to store multiple values in a single variable. They are created using square brackets [] and can hold a mix of different data types. 


let fruits = ["apple", "banana", "cherry"]; // Array of strings

let numbers = [1, 2, 3, 4, 5]; // Array of numbers

let mixed = ["apple", 1, true, null, [1, 2, 3]]; // Array of mixed types

let objects = [
  { name: "John", age: 30 },
  { name: "Jane", age: 25 }
]; // Array of objects

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]; // 2D Array (Array of Arrays)

let nestedArray = [
  ["a", "b", "c"],
  ["d", "e"],
  ["f", "g", "h", "i"]
]; // Array of Arrays with different lengths

let emptyArray = []; // Empty Array

Arrays can be accessed using their index, which starts at 0. For example, fruits[0] will return "apple".

Array Indexing

Array indexing allows you to access individual elements within an array. You can use the index number in square brackets to retrieve the value at that position. 


let fruits = ["apple", "banana", "cherry"];
let firstFruit = fruits[0]; // "apple"
let secondFruit = fruits[1]; // "banana"

Array Methods

JavaScript provides a variety of built-in methods for working with arrays. These methods allow you to manipulate and interact with array data easily. 


let fruits = ["apple", "banana", "cherry"];
fruits.push("date"); // Adds "date" to the end
fruits.pop(); // Removes the last element
fruits.shift(); // Removes the first element
fruits.unshift("kiwi"); // Adds "kiwi" to the beginning
