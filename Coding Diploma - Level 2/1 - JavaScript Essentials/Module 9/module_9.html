Conditional Statements

Conditional statements are used to perform different actions based on different conditions. In JavaScript, we primarily use the if...else statement for this purpose. 


if (condition) {
  // code to execute if condition is true
} else {
  // code to execute if condition is false
}

Example:

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

In this example, we check the age of a person and log a message to the console based on whether they are an adult or a minor. The result would be "You are an adult."
