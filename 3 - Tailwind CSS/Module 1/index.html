1.1 Introduction

Tailwind CSS is a Utility First CSS Framework for creating and styling web pages. It uses utility classes to style components within HTML.

1.2 Why We Use Tailwind?

For it's Modular Utility First Approach which allows us to have a more uniform design style and allows easier styling without ever having to write any CSS.

1.3 Setting Up Tailwind

Tailwind has multiple methods of setting up. The most commond method is using NPM however as we aren't using JavaScript in a major way, we will be using the CDN

  1. Head to the TailwindCSS Wiki
