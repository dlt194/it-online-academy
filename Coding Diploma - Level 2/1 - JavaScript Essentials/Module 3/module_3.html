A Deeper Dive into Objects

In this module, we will explore the concept of objects in JavaScript, which are collections of properties and methods. Objects are a fundamental part of JavaScript and allow for more complex data structures.

We will cover how to create objects, access their properties, and utilize methods. Additionally, we will discuss object classes and modular JavaScript practices.


Here is an example of a simple object representing some products: 

const products = [
  {
    name: "Product 1",
    price: 10,
    description: "Description of Product 1",
    image: "https://via.placeholder.com/150",
  },
  {
    name: "Product 2",
    price: 20,
    description: "Description of Product 2",
    image: "https://via.placeholder.com/150",
  },
  {
    name: "Product 3",

Object Properties & Methods

Objects in JavaScript are collections of key-value pairs, where each key is a string (or Symbol) and the value can be any data type, including other objects or functions. This allows for a flexible way to group related data and functionality together.


Properties are what describe the object, while methods define what it can do. 


const character = {
  hairColour: "brown",
  eyeColour: "blue",
  clothesColour: "red",
  goldCoins: 0,
  isDead: false,
  position: {
    x: 0,
    y: 0,
  },
  hasKey: true,
  inventory: ["sword", "shield", "potion"],
};

const chest = {
  isOpen: false,
  isLocked: false,
  coins: 10,
  inventory: ["sword", "shield", "potion"],

  openChest: function () {
    if (!this.isOpen) {
      this.isOpen = true;
      this.coins = 0;
      console.log("The chest is now open. Player collected 10 coins.");
    }
  },
};

Where do Objects Live?

Objects can be created in various scopes, such as global, function, or block scope. They can also be nested within other objects or arrays, allowing for complex data structures.


In the example above, the 'character' and 'chest' objects are defined in the global scope, making them accessible throughout the script.


Accessing Objects and their Properties

In JavaScript when we want to work with an object, all we need to do is call it's name (once it has been defined).


For example

const character = {
  hairColour: "brown",
  eyeColour: "blue",
  clothesColour: "red",
  goldCoins: 0,
  isDead: false,
  position: {
    x: 0,
    y: 0,
  },
  hasKey: true,
  inventory: ["sword", "shield", "potion"],
};

console.log(character.hairColour); // Outputs: brown

In the example above, we access the 'hairColour' property of the 'character' object using dot notation. We can also use bracket notation to access properties dynamically.

Object Classes

Object classes are relatively new to JavaScript. Because we want to have the least amount of code repitition as possible, we can create a class that defines the properties and methods of an object. This allows us to create multiple instances of the same object type without repeating code. 


class Character {
  constructor(name, health, hairColour, clothesColour, goldCoins) {
    this.name = name;
    this.health = health;
    
    this.hairColour = hairColour;
    this.clothesColour = clothesColour;
    this.goldCoins = goldCoins;
  }

// New Character Instances
const player = new Character("Hero", 100, "brown", "blue", 0);
const enemy = new Character("Villain", 80, 40, "black", "red", 10);
}

We can access the properties of the class instances just like we would with a normal object. Using dot notation, we can access the specific properties of the instances we created. 


console.log(player); // Outputs: Character { name: 'Hero', health: 100, hairColour: 'brown', clothesColour: 'blue', goldCoins: 0 }
console.log(enemy.hairColour); // Outputs: black

Object Class Methods

Object classes can also have methods defined within them. This allows us to encapsulate functionality related to the object within the class itself. 


class Chest {
  constructor(isOpen, coins) {
    this.isOpen = isOpen;
    this.coins = coins;
  }

  openChest() {
    if (!this.isOpen) {
      this.isOpen = true;
      console.log("The chest is now open. Player collected " + this.coins + " coins.");
      this.coins = 0; // Reset coins after opening
    } else {
      console.log("The chest is already open.");
    }
  }
}

const myChest = new Chest(false, 10);

myChest.openChest(); // Outputs: The chest is now open. Player collected 10 coins.

In the example above, the 'Chest' class has a method called 'openChest' that allows us to open the chest and collect coins. This encapsulates the functionality related to the chest within the class itself.
