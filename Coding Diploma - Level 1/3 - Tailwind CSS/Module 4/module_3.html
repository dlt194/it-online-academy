3.1 Layouts and Components

Flexbox & Grids function the same as they do in normal CSS however, they use slightly different syntax

Flexbox

<div class="flex">
  <div class="flex-1"></div>
  <div class="flex-2"></div>
  <div class="flex-3"></div>
  <div class="flex-4"></div>
</div>

Grid with 3 Cols

<div class="grid grid-cols-3">
  <div></div>
  <div></div>
  <div></div>
</div>

Grid with 3 Rows

<div class="grid grid-rows-3">
  <div></div>
  <div></div>
  <div></div>
</div>

3.2 Creating a Navbar

A navbar is a common component in web design. It allows users to navigate between different sections of a website. In Tailwind CSS, we can create a simple navbar using utility classes.

Navbar

<div class="flex justify-center">
  <a href='#' class='m-1 p-1 rounded-md bg-green-400 hover: underline hover:bg-blue-500'>Home</a>
  <a href='#' class='m-1 p-1 rounded-md bg-green-400 hover: underline hover:bg-blue-500'>About</a>
  <a href='#' class='m-1 p-1 rounded-md bg-green-400 hover: underline hover:bg-blue-500'>Contact</a>
  <a href='#' class='m-1 p-1 rounded-md bg-green-400 hover: underline hover:bg-blue-500'>Gallery</a>
</div>

What is the Config?

It is for configuring the settings of your programme. In Tailwind we can use our config to futher customise our utilites and components.

Config

<script>
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'clifford': '#da373d',
            },
        },
    },
};
</script>

<div class="bg-clifford text-white"></div>
