Objects, Methods, Properties & Functions

3 of the most common objects in JavaScript are:

  • Window
  • Document
  • Console

Window is the global object in JavaScript, it represents the browser window. The document object represents the HTML document loaded in the browser, and the console object provides access to the browser's debugging console.

To access the window object, you can use the `window` keyword.


window.alert("Hello, World!");

This code will display an alert box with the message "Hello, World!" in the browser.




The window object has various properties. A good example of this would be onload, which is an event handler that triggers when the page is loaded. 


window.onload = function() {
  console.log("Page is loaded!");
};

This code will log "Page is loaded!" to the console when the page finishes loading.

This would be considered an anonymous function. This is because it does not have a defined name, it runs in the background.

Variables & Data Types

Variables are used to store data in JavaScript. They can hold different types of values, such as numbers, strings, booleans, and objects. In JavaScript, you can declare variables using the 'var', 'let', or 'const' keywords.


  • var: Used to declare a variable that can be reassigned later. It has function scope.
  • let: Used to declare a block-scoped variable that can be reassigned. It is preferred over 'var' for block scope.
  • const: Used to declare a block-scoped variable that cannot be reassigned. It is used for constants. 

var myAge;
var myAge = 30;
var x = 10;
var charaterName = "Sponge";
var isRaining = false;

let myAge;
let myAge = 30;
let x = 10;
let charaterName = "Sponge";
let isRaining = false;

const myAge;
const myAge = 30;
const x = 10;
const charaterName = "Sponge";
const isRaining = false;
