What is JavaScript?

JavaScript is a client-side programming language that is used to create functionality and control the user interactions in applications.


  • HTML & CSS defines the structure, content and styling of a webpage.
  • JavaScript then can control the HTML and CSS to make it more dynamic.
  • JavaScript can add programming cpde and logic to define the behavior of the web page.

  • JavaScript is free
  • We do not have to download or install it, JavaScript is installed in most browsers.

Different Ways of Using JavaScript

We can look at using JavaScript in different ways:

  • Use built-in functions and methods
  • Use programming logic and algorithms to create custom solutions and functionality.
  • Use predefined frameworks and libraries.
  • Use combinations of these approaches.

How Do We Use JavaScript?

JavaScript is usually used for webpages and is ised and executed by web browsers.

  • Inline - It can be embedded into the HTML document using the <script></script> tags.
  • External - It can be added using an external .js file that are linked into the HTML web pages using the <script> element with the "src" attribute.
