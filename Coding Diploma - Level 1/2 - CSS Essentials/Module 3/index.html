Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

                h1 {
                    color: blue;
                    font-size: 12px;
                }
  • The selector points to the HTML Element you wish to style
  • The declaration block contains one or more declarations separated by semicolons
  • Each declaration includes a CSS property name and a value, separated by a colon
  • Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Selectors

CSS Selectors are used to "find" (or select) the HTML elements you want to style. We can divide CSS selectors into 5 categories:

  • Simple selectors - select elements based on name, id class. 
    e.g.,
            Element: p {
                text-align: center;
                color: red
                }.
                
    
            ID: #id {
                text-align: center;
                color: red
                }, 
                
    
            Class: .class {
                text-align: center;
                color: red
                }

  • Psuedo-class selectors - select elements based on certain state. (e.g., )
  • Psuedo-elements selectors - select and style a part of an element. (e.g., h1)
  • Attribue selectors - select elements based on an attribute or attribute value. (e.g., h1)

Properties

CSS properties define the styles to be applied. Examples include:

  • color: Sets the text color.
  • background-color: Sets the background color.
  • font-size: Sets the size of the text.
