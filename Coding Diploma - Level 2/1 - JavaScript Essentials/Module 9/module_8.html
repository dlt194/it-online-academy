Introduction to Events

Events are actions or occurrences that happen in the browser, such as clicks, key presses, or mouse movements. JavaScript can respond to these events and execute code when they occur. 


document.addEventListener('click', function(event) {
  console.log('Document clicked!', event);
});

In this example, we add an event listener to the document that listens for click events. When the document is clicked, it logs a message to the console.


Event handling is important for creating interactive web applications. It allows developers to respond to user actions, such as clicks, key presses, and mouse movements.

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');
const circles = [];

canvas.width = window.innerWidth;
canvas.height = window.innerHeight;

function Circle(x, y, radius) {
  this.x = x;
  this.y = y;
  this.radius = radius;
  this.color = color;

  this.draw = function() {
    ctx.beginPath();
    ctx.arc(this.x, this.y, this.radius, 0, Math.PI * 2);
    ctx.fillStyle = this.color;
    ctx.fill();
  }
}

canvas.addEventListener('mousemove', function(event) {
  const radius = Math.random() * 30 + 10;
  const color = `rgb(${Math.random() * 255}, ${Math.random() * 255}, ${Math.random() * 255})`;
  circles.push(new Circle(event.clientX, event.clientY, radius, color));
})

canvas.addEventListener('click', function(event) {
  const radius = Math.random() * 30 + 10;
  const color = `rgb(${Math.random() * 255}, ${Math.random() * 255}, ${Math.random() * 255})`;
  circles.push(new Circle(event.clientX, event.clientY, radius, color));
});

function animate() {
  requestAnimationFrame(animate);
  ctx.clearRect(0, 0, canvas.width, canvas.height);

  circles.forEach(circle => {
    circle.draw();
  });
}

This code creates a canvas where circles are drawn at the mouse position when the mouse moves or clicks. Each circle has a random radius and color.


Example

Add Event Listener Example

This example demonstrates how to add an event listener to a button element. When the button is clicked, it logs a message to the console. 


const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  alert('Button clicked!');
});

In this example, we select a button element with the ID "myButton" and add an event listener that listens for click events. When the button is clicked, it displays an alert message.


Event listeners can be added to various elements, such as buttons, links, and even the document itself. They allow developers to respond to user interactions and create dynamic web applications.


For more information on event listeners, you can refer to the MDN Web Docs - addEventListener.
