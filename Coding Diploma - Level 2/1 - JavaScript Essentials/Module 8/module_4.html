Template Literals

Template literals are string literals allowing embedded expressions. They can be multi-line and include variables and expressions using the `${expression}` syntax. 


            
const name = 'Dan';
const greeting = `Hello, ${name}! Welcome to JavaScript Essentials.`;
console.log(greeting); // Output: Hello, Dan! Welcome to JavaScript Essentials.

Template literals are enclosed by backticks (``) instead of single or double quotes, allowing for easier string interpolation and multi-line strings. 


            
// script.js
class User {
  constructor(name, age, email) {
    this.name = name;
    this.age = age;
    this.email = email;
  }
}

const userObject = new User("Dan", 30, "dan@example.com");

console.log(userObject);


const newHTML = `
<h1>${userObject.name}</h1>
<h1>${userObject.age}</h1>
<h1>${userObject.email}</h1>
`;

document.body.innerHTML = newHTML;

// index.html
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <script src="script.js"></script>
</body>
</html>

In this example, we create a `User` class and instantiate it with some data. We then use template literals to create an HTML string that includes the user's name, age, and email, which is then inserted into the document body.

String Concatenation

String concatenation is the process of joining two or more strings together. In JavaScript, this can be done using the `+` operator or template literals. 


     
const firstName = 'Dan';
const lastName = 'Thomas';
const fullName = firstName + ' ' + lastName;
console.log(fullName); // Output: Dan Thomas

In this example, we concatenate `firstName` and `lastName` using the `+` operator to create a full name.


Alternatively, you can use template literals for cleaner syntax: 


            
const fullName = `${firstName} ${lastName}`;
console.log(fullName); // Output: Dan Thomas

This method is more readable and allows for easier inclusion of variables and expressions within strings.

Finding Elements in the DOM

The Document Object Model (DOM) represents the structure of a web page. You can find elements in the DOM using various methods such as `getElementById`, `getElementsByClassName`, and `querySelector`. 


document.getElementById('myElement'); // Finds an element by its ID
document.getElementsByClassName('myClass'); // Finds elements by class name
document.querySelector('.myClass'); // Finds the first element matching the selector
document.querySelectorAll('.myClass'); // Finds all elements matching the selector
document.querySelector("nav li:last-child"); // Finds the last child of a nav element

document.querySelector("nav li:last-child")
.forEach((item) => item.style.backgroundColor = "red"); // Changes the background color of the last nav item

These methods allow you to access and manipulate elements in the DOM easily.

Get Elements & Classes by Id

To get an element by its ID, you can use the `getElementById` method. This method returns the first element with the specified ID. 


document.getElementById('myElement'); // Returns the element with ID 'myElement'

To get elements by class name, you can use the `getElementsByClassName` method, which returns a live HTMLCollection of elements with the specified class. 


document.getElementsByClassName('myClass'); // Returns elements with class 'myClass'

You can also use `querySelector` to get the first element that matches a CSS selector, or `querySelectorAll` to get all elements that match a selector. 


document.querySelector('#myElement'); // Returns the first element with ID 'myElement'
document.querySelectorAll('.myClass'); // Returns all elements with class 'myClass'

These methods are essential for manipulating the DOM and accessing elements based on their IDs or classes. 


// script.js
document.addEventListener("DOMContentLoaded", function() {
  const chapterSection = document.getElementById('chapter-section-id');
  chapterSection.style.backgroundColor = "gold";
  chapterSection.style.padding = "20px";

  const paragraph = document.getElementById("paragraph-id");
  paragraph.style.fontSize = "20px";
  paragraph.style.color = "blue";
  paragraph.style.fontWeight = "bold";

Manipulating Classes

You can manipulate classes of elements using the `classList` property. This property provides methods to add, remove, and toggle classes on an element. 


const element = document.getElementById('myElement');
element.classList.add('newClass'); // Adds a class
element.classList.remove('oldClass'); // Removes a class
element.classList.toggle('toggleClass'); // Toggles a class
element.classList.contains('checkClass'); // Checks if a class exists

These methods allow you to dynamically change the classes of an element, which can be useful for styling and behavior changes.

Attribute Properties & Methods

Attributes in HTML elements can be accessed and modified using the `getAttribute`, `setAttribute`, and `removeAttribute` methods. 


const element = document.getElementById('myElement');
element.getAttribute('data-custom'); // Gets the value of a custom attribute
element.setAttribute('data-custom', 'newValue'); // Sets a new value for a custom attribute
element.removeAttribute('data-custom'); // Removes a custom attribute

These methods allow you to interact with the attributes of HTML elements, enabling dynamic changes to the element's properties.


You can also select an element using attributes with `hasAttribute` 


document.querySelector("img").hasAttribute("alt"); // Checks if the image has an alt attribute

This method returns `true` if the specified attribute exists on the element, allowing you to conditionally manipulate elements based on their attributes.

Inline Style Property

The inline style property allows you to directly manipulate the CSS styles of an element using JavaScript. You can access and modify the `style` property of an element to change its appearance. 


const element = document.getElementById('myElement');
element.style.color = 'red'; // Changes the text color to red
element.style.backgroundColor = 'blue'; // Changes the background color to blue
element.style.fontSize = '20px'; // Changes the font size to 20px
element.style.display = 'none'; // Hides the element
element.style.display = 'block'; // Shows the element

This method allows you to dynamically change the styles of an element, providing a powerful way to control the appearance of your web page.

Adding Element to the DOM

To add an element to the DOM, you can create a new element using `document.createElement`, set its attributes and content, and then append it to an existing element using `appendChild` or `insertBefore`. 


const newElement = document.createElement('div'); // Creates a new div element
newElement.id = 'newElementId'; // Sets an ID for the new element
newElement.className = 'newElementClass'; // Sets a class for the new element
newElement.textContent = 'This is a new element'; // Sets the text content
const parentElement = document.getElementById('parentElementId'); // Gets the parent element
parentElement.appendChild(newElement); // Appends the new element to the parent

This method allows you to dynamically create and insert new elements into the DOM, enabling you to build interactive and dynamic web applications. 


window.onLoad = addElement;

function addElement() {
  const newSection = document.createElement('section');

  const newContent = document.createTextNode('This is a new section added to the DOM.');

  newSection.appendChild(newContent);

  const currentSection = document.querySelector('#chapter-section-id');
  document.body.insertBefore(newSection, currentSection);
}

In this example, we create a new section element, add text content to it, and insert it before an existing section in the document.
